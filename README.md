# PersistentStaminaMod

______________________________________________________________________English

PersistentStaminaMod is a BepInEx plugin for **Burglin’ Gnomes** that allows:
- Infinite stamina toggle (F5)  
- Strong arms for heavy objects (F6)  
- Unlimited arm reach (F7)  
- All features configurable in-game via a config file

Installation
1. Make sure **BepInEx 5.4.23.5** is installed in your game folder.  
2. Copy `StaminaMod.dll` to the `BepInEx/plugins` folder:  <GameFolder>\BepInEx\plugins\
3. Launch the game.  
4. Configuration file will appear at: <GameFolder>\BepInEx\config\StaminaMod.cfg
5. Use the hotkeys in-game:  
- **F5** → Toggle infinite stamina  
- **F6** → Enable strong arms  
- **F7** → Remove arm reach limit  

Requirements
- Burglin’ Gnomes compatible with BepInEx 5.4.x  
- BepInEx 5.4.23.5

**PersistentStaminaMod** — плагин для **Burglin’ Gnomes**, который позволяет:
- Бесконечная стамина (F5)  
- Сильные руки для тяжёлых предметов (F6)  
- Неограниченная длина рук (F7)  
- Все функции настраиваются через конфиг-файл

Установка
1. Убедитесь, что **BepInEx 5.4.23.5** установлен в папку с игрой.  
2. Скопируйте `StaminaMod.dll` в папку `BepInEx/plugins`: <Папка с игрой>\BepInEx\plugins\
3. Запустите игру.  
4. Файл конфигурации появится по пути:  <Папка с игрой>\BepInEx\config\StaminaMod.cfg
5. Горячие клавиши в игре:  
**F5** → Включить/выключить бесконечную стамину  
**F6** → Включить сильные руки  
**F7** → Убрать ограничение длины рук  

- Burglin’ Gnomes  (совместимая с BepInEx 5.4.x)  
- BepInEx 5.4.23.5
